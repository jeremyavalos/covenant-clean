# Covenant Beta Test Checklist

Use this checklist on a real device with the latest approved beta build.

- [ ] Install Covenant from TestFlight or Android internal testing.
- [ ] Open the app successfully.
- [ ] Register with a real email address.
- [ ] Receive the verification email.
- [ ] Verify the email address.
- [ ] Log in with the verified account.
- [ ] Choose one free habit.
- [ ] Open the selected habit.
- [ ] Complete the habit for today.
- [ ] Close and reopen the app.
- [ ] Confirm progress persistence: completed day, streak, and 30-day progress should remain visible.
- [ ] Enter deeper from the habit screen.
- [ ] Return from deeper to the app flow.
- [ ] Open the paywall.
- [ ] Test restore purchases.
- [ ] Log out.
- [ ] Log in again with the same account.
- [ ] Confirm progress is still present after logging in again.
- [ ] Report any bug with device model, OS version, build number, screenshots or screen recording, and exact reproduction steps.
